HTML Text Formatting Tags

Tag Description Example
<b> or <strong> Bold text This text is bold or strong
<i> or <em> Italic text This text is italic or emphasized
<u> Underline text This text is underlined: important
<strike> Strikethrough text (deprecated) This text is strikethrough (use CSS instead)
<sup> Superscript text x2 (exponent)
<sub> Subscript text H2O (water molecule)
<mark> Highlighted text This text is highlighted
